# Sample Attributes
The Style for describing a Person/Entity...

>Normal:

```
- Name: John Doe
- Age: 25
- Friends: Joseph, Emily, Sarah
- Occupation: Software Engineer
- Hobbies: Reading, Hiking, Gaming
```

>**BOLD**:

```
- **Name:** John Doe
- **Age:** 25
- **Friends:** Joseph, Emily, Sarah
- **Occupation:** Software Engineer
- **Hobbies:** Reading, Hiking, Gaming
```
